# Finance & Compliance Performance Assessment

## Business Context
This project simulates a real-world finance and compliance assessment where the objective was to evaluate financial accuracy, cost controls, and performance reporting quality. The work mirrors analyst responsibilities in consulting and financial services environments, with a strong focus on structured analysis, data validation, and decision-ready insights.

## Objectives
- Evaluate the accuracy and structure of P&L reporting
- Identify cost leakages through variance analysis
- Validate financial data for consistency and reliability
- Assess compliance risks in financial reporting
- Deliver clear insights to support management decision-making

## Dataset & Inputs
### Data Used
- P&L statement data (revenue, COGS, operating expenses)
- Vendor-level and cost center expense data
- Forecast vs actual financial figures

### Data Handling
- Data cleaning and standardization in Excel
- Reconciliation checks across financial statements
- Validation rules to identify anomalies and discrepancies

## Methodology
### Analytical Approach
1. Structured P&L breakdown to understand revenue and cost drivers
2. Variance analysis comparing actuals vs planned figures
3. Identification of cost leakages and control gaps
4. Performance benchmarking across expense categories
5. Compliance-focused review of financial accuracy and reporting structure

## Key Insights
- Improved margin visibility through structured P&L presentation
- Identified expense classification issues impacting reporting accuracy
- Highlighted areas requiring stronger financial controls
- Strengthened confidence in financial data through validation checks

## Deliverables
- Excel-based financial model
- Variance analysis tables and summaries
- Management-ready insights for finance and compliance review
